Personal dotfiles of my hyprland setup.

## How do I install ts
It's just a couple of commands
```
rm -rf ~/.config/hypr # Git won't let you clone in a non-empty directory.
git clone https://github.com/j3h1/dot-hypr ~/.config/hypr
```
